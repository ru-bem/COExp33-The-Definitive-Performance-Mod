::---- GI:           High - Decrease for more performance       ::
::---- Reflection:   Epic                                       ::
::---- Post Process: Low                                        ::
::---- Texture:      Epic                                       ::
::---- Visual FX:    High                                       ::

::---- Shading:      High                                       ::
